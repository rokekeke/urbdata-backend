# ADR 006 - Parametros padrao

- Status: parcial
- Data: 2026-07-16
- Aprovador de dominio: pendente

Parametros sao tipados e centralizados, podem ser sobrescritos em testes e devem ser copiados para `analysis_run.config`. Mapeamentos urbanisticos e valores ainda nao aprovados permanecem vazios ou nulos; nenhuma funcao recebe um valor de dominio inventado.
