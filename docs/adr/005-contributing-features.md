# ADR 005 - Feicoes contribuintes

- Status: aceito
- Data: 2026-07-16
- Aprovador de dominio: pendente

O indice do GeoDataFrame nunca sera ID de negocio. Camadas carregadas preservam `feature_id`, e resultados registram as feicoes contribuintes separadamente da futura feicao objeto do resultado. Limites de cardinalidade deverao ser configuraveis.
